# Apollo Hospital Prescription Management System

A web-based healthcare platform developed for Apollo Hospital Visakhapatnam.

## Features
- Doctor Dashboard
- Patient Dashboard
- Prescription Form
- Drug Search
- Dosage Validation
- Refill Requests
- Prescription History
- Notifications
- Admin Panel

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Node.js
- MySQL
